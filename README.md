# 🧭 hermes-ui - Control Hermes from one screen

## 🚀 What this app does

hermes-ui gives you one place to work with Hermes Agent. You can chat with it, guide it, open files, manage skills, and watch what it is doing.

It runs as a simple HTML app with a glass-style interface. You use it in your browser, so it feels light and easy to open.

## 🖥️ What you need

- Windows 10 or Windows 11
- A web browser such as Edge, Chrome, or Firefox
- A working copy of Hermes Agent
- A stable internet or local network link to your Hermes setup

If you use Tailscale, you can connect to a remote Hermes Agent on your private network.

## 🧭 First run

When you open hermes-ui for the first time, it may ask for your Hermes Agent address.

Use the address where your agent runs. This may be:

- A local address on your PC
- A network address on your home setup
- A Tailscale address for remote access

After you connect, the app shows the main dashboard. From there, you can start chat, browse files, and check system status.

## 💬 Chat with Hermes

The chat view lets you talk to Hermes Agent in plain language.

Use it to:

- Ask questions
- Give tasks
- Check progress
- Review answers
- Keep a history of what you asked

This is the main place to steer Hermes while it works.

## 📁 Browse files

The file view helps you look through folders and files that Hermes can access.

You can use it to:

- Open project files
- Check output files
- Find logs
- Review documents
- Move between folders fast

This is useful when you want to see what Hermes changed or created.

## 🛠️ Manage skills

Skills let Hermes do specific tasks.

From the skills area, you can:

- Turn skills on or off
- See what each skill does
- Add new skills if your setup supports it
- Remove skills you no longer need
- Keep your workflow tidy

If you use more than one skill, this view helps you stay in control.

## 📡 Monitor activity

The dashboard shows what Hermes is doing right now.

You can watch:

- Active tasks
- Status messages
- File changes
- Agent state
- Connection health

This helps you know if Hermes is working, waiting, or needs attention.

## 🔗 Connect over Tailscale

If your Hermes Agent runs on another machine, Tailscale can make the connection easier.

Typical use:

1. Install and sign in to Tailscale on both machines.
2. Make sure both devices can see each other.
3. Copy the Tailscale address of the machine running Hermes Agent.
4. Paste that address into hermes-ui.
5. Save the setting and refresh the app.

This works well when you want to reach Hermes from another room or another network.

## 🎨 Interface guide

You will usually see:

- A main chat area

The layout aims to keep the most used actions close together, so you do not have to jump around.

## 🔄 Typical workflow

A common way to use the app is:

1. Open hermes-ui in your browser.
2. Connect to Hermes Agent.
3. Start a chat request.
4. Watch the task status.
5. Open files when you need to review results.
6. Adjust skills if the task needs a different tool.
7. Keep an eye on the dashboard until the work is done

This flow keeps everything in one place.

## 🧱 Basic setup tips

- Keep the app and Hermes Agent on the same network when possible
- Use a Tailscale link if you need remote access
- Keep your release file in a folder you can find again
- If the page does not load, check the agent address first
- Refresh the browser after changing connection details

## 📝 File layout

If you open the app folder, you may see files such as:

- `index.html`
- `assets`
- `config`
- `readme`
- `icons`

Keep the folder together if the release comes as a zip. The app may need more than one file to load.

## 🧪 If something looks wrong

If the app does not connect, check these items:

- The Hermes Agent is running
- The address is correct
- Your browser can reach the machine
- Tailscale is signed in on both devices
- The release file was unpacked fully if it came in a zip

If the screen is blank, reload the page and try again after checking the file path.
